Vista Previa del Proyecto

SalvaVidas es una aplicación diseñada para ayudar a profesionales de la salud mental en la detección y prevención del riesgo suicida en pacientes, mediante el análisis de datos clínicos asistido por inteligencia artificial.

Chat con IA

Interfaz de conversación intuitiva para consultar sobre casos y recibir recomendaciones de intervención.

Evaluación de Riesgo

Análisis de factores de riesgo y señales de alerta, con clasificación por nivel de gravedad.

Guías de Intervención

Generación de guías personalizadas para el abordaje terapéutico y seguimiento de pacientes.

Cómo acceder a la aplicación

Para acceder correctamente a la aplicación en un entorno de desarrollo local, sigue estos pasos:

  1. Asegúrate de que Apache y MySQL estén funcionando en XAMPP
  2. La URL completa para acceder a la aplicación debería ser similar a:
    http://localhost/salvavidasapp/public/
  3. Para acceder a rutas específicas, usa el formato:
    http://localhost/salvavidasapp/public/index.php/salvavidasapp/dashboard

Solución de Problemas Comunes

Error 404 al acceder a las rutas: Esto puede deberse a la configuración del servidor Apache. Prueba con las siguientes soluciones:

  • Asegúrate de que el módulo rewrite de Apache esté habilitado
  • Verifica que los archivos .htaccess estén correctamente configurados
  • Intenta acceder directamente a través de index.php en lugar de rutas amigables
